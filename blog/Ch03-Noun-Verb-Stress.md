# 임용 음운론 Ch.3 Noun-Verb Stress 완전 정리: 품사로 갈리는 강세 이동·penult 규칙·vowel reduction

같은 철자 `record`가 명사면 첫 음절, 동사면 둘째 음절에 강세가 온다. 이렇게 <span style="color:#3182ce;">**품사(part of speech)가 강세 위치를 바꾸는 현상**</span>이 이 챕터의 핵심이다. 2음절어에서 명사·형용사는 앞 음절, 동사는 뒤 음절로 강세가 가는 경향을 정확히 일반화하고, <span style="color:#3182ce;">**penult syllable weight**</span>와 <span style="color:#3182ce;">**vowel reduction**</span>이 어떻게 맞물리는지까지 정리한다. 흐름은 **① 개념 정의 → ② 통계적 경향 → ③ 판단 포인트·derivation → ④ 함정·수험 팁** 4단계다.

---

## ① 개념 정의 — 품사가 강세를 가른다

영어 2음절어 중에는 철자가 같고 품사만 다른 짝이 있다. 이때 강세 위치로 명사와 동사를 구분한다.

<span style="color:#3182ce;">**Noun-verb stress alternation**</span>의 기본 일반화는 단순하다.

- 2음절 <span style="color:#3182ce;">**명사·형용사**</span>는 주로 <span style="color:#3182ce;">**penult(2음절어에서는 첫 음절)**</span>에 강세.
- 2음절 <span style="color:#3182ce;">**동사**</span>는 주로 <span style="color:#3182ce;">**ult(마지막 음절)**</span>에 강세.

대표 minimal pair를 IPA로 보면 강세 위치 차이가 바로 드러난다.

| 단어 | 명사 (penult) | 동사 (ult) |
|------|--------------|-----------|
| record | ˈrecord /ˈrekɔːrd/ | reˈcord /rɪˈkɔːrd/ |
| permit | ˈpermit /ˈpɜːrmɪt/ | perˈmit /pərˈmɪt/ |
| conduct | ˈconduct /ˈkɒndʌkt/ | conˈduct /kənˈdʌkt/ |
| object | ˈobject /ˈɒbdʒɪkt/ | obˈject /əbˈdʒekt/ |
| protest | ˈprotest /ˈproʊtest/ | proˈtest /prəˈtest/ |
| increase | ˈincrease /ˈɪnkriːs/ | inˈcrease /ɪnˈkriːs/ |
| insult | ˈinsult /ˈɪnsʌlt/ | inˈsult /ɪnˈsʌlt/ |
| refund | ˈrefund /ˈriːfʌnd/ | reˈfund /rɪˈfʌnd/ |

<span style="color:#805ad5;">**Yavaş**</span>의 기술에 따르면 이 대립은 형용사에도 그대로 적용된다. <span style="color:#319795;">**ˈpresent**</span>는 명사·형용사일 때 첫 음절, <span style="color:#319795;">**preˈsent**</span>는 동사일 때 둘째 음절이다. 즉 강세를 "동사만 뒤로 민다"가 아니라 <span style="color:#3182ce;">**명사·형용사 = 앞 / 동사 = 뒤**</span> 구도로 외우는 것이 정확하다.

---

## ② 통계적 경향 — 규칙이 아니라 tendency

이 현상에서 가장 중요한 판단은 "절대 규칙이 아니라 강한 경향"이라는 점이다. <span style="color:#805ad5;">**Yavaş**</span>가 제시한 수치를 그대로 기억해 두면 선지 판단이 빨라진다.

- 2음절 **명사**: 약 <span style="color:#3182ce;">**81.7%**</span>가 penult(첫 음절) 강세.
- 2음절 **형용사**: 약 <span style="color:#3182ce;">**81%**</span>가 penult 강세. (e.g. ˈactive, ˈcommon, ˈready)
- 2음절 **동사**: 약 <span style="color:#3182ce;">**60% 이상**</span>이 ult(마지막 음절) 강세.

여기서 두 가지가 보인다. 첫째, 명사·형용사의 penult 경향(80%대)이 동사의 ult 경향(60%대)보다 더 강하다. 둘째, 동사의 40% 가까이는 이 경향을 따르지 않는다 — 즉 <span style="color:#c53030;">**동사라고 무조건 뒤 음절 강세는 아니다**</span>. 그래서 "예외"가 출제 포인트가 된다(④ 참조).

---

## ③ 판단 포인트 — penult 무게와 vowel reduction

### penult이 가벼우면 강세가 뒤로 간다

강세 위치를 단순히 "품사"로만 외우면 한 단계가 빠진다. 실제 영어 강세는 <span style="color:#3182ce;">**penult syllable weight(음절 무게)**</span>에 민감하다. 핵심 원리는 이렇다.

- penult이 <span style="color:#3182ce;">**heavy**</span>(장모음·이중모음 또는 폐음절)면 → penult이 강세를 끈다.
- penult 모음이 <span style="color:#3182ce;">**/ə/**</span>처럼 weightless면 → penult이 강세를 못 끌고 → 강세가 <span style="color:#3182ce;">**ult로 이동**</span>한다.

그래서 명사인데도 둘째 음절에 강세가 오는 경우가 생긴다.

- <span style="color:#319795;">**appéal**</span> /əˈpiːl/ — 첫 음절 모음이 /ə/라 강세를 못 받음 → ult로.
- <span style="color:#319795;">**ballóon**</span> /bəˈluːn/ — 같은 원리로 둘째 음절 강세.

즉 명사·동사 강세 대립은 더 큰 강세 체계(penult 무게 기반)의 한 단면이다. 3음절 이상 단어는 이 무게 규칙이 본격적으로 작동하므로, noun-verb 짝의 "품사 일반화"를 그대로 확장하면 안 된다.

### 강세 이동은 vowel reduction을 동반한다

강세가 빠진 음절의 모음은 흔히 <span style="color:#3182ce;">**/ə/로 reduction**</span>된다. minimal pair에서 이 변화를 같이 봐야 답안이 정확해진다.

```
record (N): /ˈrekɔːrd/   — 1음절 강세, 2음절 모음 약화
record (V): /rɪˈkɔːrd/   — 강세가 2음절로, 1음절 모음이 /ɪ/로 약화
```

```
object (N): /ˈɒbdʒɪkt/   — 1음절 강세, 2음절 /ɪ/
object (V): /əbˈdʒekt/   — 강세가 2음절로, 1음절이 /ə/로 약화
```

<span style="color:#dd6b20;">**쌤 팁**</span> — 답안에 강세 위치만 적지 말고 "강세 이동 + 비강세 음절의 vowel reduction"을 함께 기술하면 분석이 완성된다. 강세는 위치 문제이자 모음 질(quality) 문제다.

---

## ④ 함정 정리 + 수험 팁

### 함정 1 — 모든 2음절어가 갈리지 않는다

<span style="color:#c53030;">**⚠️ noun-verb 강세 이동은 tendency다.**</span> 동사인데도 첫 음절 강세를 유지하는 단어가 있다.

- <span style="color:#c53030;">**love, show, call**</span> 등은 명사·동사 모두 첫 음절 강세 — 이동 없음.

"동사 = 무조건 뒤 음절"로 일반화한 선지는 이런 단어에서 깨진다.

### 함정 2 — increase는 이동이 "있다"

<span style="color:#c53030;">**⚠️**</span> increase 같은 단어를 "강세 이동이 없다"고 처리하면 오답이다. <span style="color:#319795;">**ˈincrease(N) vs inˈcrease(V)**</span>로 분명히 갈린다. 평소 발음이 굳어 있어 헷갈리기 쉬운 짝(increase, protest, refund)을 따로 점검해 두자.

### 함정 3 — IPA 강세 기호 위치

<span style="color:#c53030;">**⚠️**</span> 강세 기호 <span style="color:#3182ce;">**ˈ**</span>는 강세 음절 **앞**에 붙인다. object 동사는 /əbˈdʒekt/처럼 둘째 음절 앞에 기호가 온다. /ˈobject/와 /obˈject/를 반대로 적는 실수가 잦다.

### 함정 4 — 품사 일반화의 한계

<span style="color:#c53030;">**⚠️**</span> noun-verb 패턴은 2음절어 중심의 경향이다. 3음절 이상에서는 penult syllable weight 규칙이 우선하므로, "명사니까 앞, 동사니까 뒤"를 기계적으로 확장하면 틀린다.

### 복습 포인트

<span style="color:#dd6b20;">**현직쌤 팁**</span> — noun-verb 강세 문항은 세 단계로 정리하면 안 틀린다. ① 품사를 확정(명사·형용사 → penult / 동사 → ult 경향), ② penult이 가벼운지(/ə/ 등) 확인해 강세가 뒤로 갈 가능성 점검, ③ 강세 이동에 따른 vowel reduction을 IPA에 반영. 마지막으로 love·show 같은 예외와 increase·protest 같은 "이동 있는" 짝을 한 번 더 떠올린다. 강세는 위치이자 모음 질이라는 점을 답안에 같이 적으면 분석이 완성된다.
