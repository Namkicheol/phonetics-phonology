음운론 기출에서 **Stress** 단원은 단독 출제 빈도가 높고, 특히 <span style="color:#3182ce;">**N-V Homograph Pairs**</span>와 <span style="color:#3182ce;">**Derivation Chains**</span>는 거의 매년 어딘가에서 등장한다. 이 글은 서브노트에 정리된 핵심 규칙들을 **① 개념 정의**(영어 원문 + 한글 풀이) → **② 기출 맥락** → **③ 키텀 비교** → **④ 수험 팁** 4섹션으로 풀어낸다.

---

## ① 개념 정의 — 핵심 규칙을 영어 원문과 함께

### 1) Syllable Weight — 강세를 결정하는 토대

> A syllable is **heavy** if its rhyme branches — that is, it contains a long vowel, a diphthong, or a short vowel followed by a coda consonant. A **light** syllable has a non-branching rhyme (CV structure or schwa). Schwa [ə] is **weightless** and can never bear primary stress.

영어 강세의 핵심 원리는 <span style="color:#3182ce;">**syllable weight**</span>다. rhyme이 branching하면 <span style="color:#3182ce;">**heavy syllable**</span>, non-branching이면 <span style="color:#3182ce;">**light syllable**</span>. heavy syllable만 primary stress를 받을 수 있다. 강세 위치를 기술할 때는 뒤에서부터 세는 음절 위치 용어를 쓴다: <span style="color:#3182ce;">**ultimate (ult)**</span> = 마지막 음절, <span style="color:#3182ce;">**penultimate (penult)**</span> = 끝에서 둘째, <span style="color:#3182ce;">**antepenultimate (antepenult)**</span> = 끝에서 셋째.

> **Schwa [ə] is weightless** — no matter where it appears, it cannot receive primary stress. When the penult contains schwa, stress automatically shifts to the antepenult.

<span style="color:#c53030;">**핵심 포인트**</span>: penult에 schwa가 오면 penult 강세가 불가능하므로 antepenult로 강세가 이동한다. 이것이 3음절 이상 영어 단어 강세의 핵심 메커니즘이다.

| 구분 | 조건 | 예 |
|------|------|----|
| Heavy | 장모음·이중모음 / 단모음 + coda | a**gen**da [dʒɛn] — closed |
| Light | 단모음 + coda 없음 (CV) | **ca**lendar [kəl] — penult = ə |
| Weightless | schwa [ə] | **a**nimal — penult [məl] → ə |

> 💡 <span style="color:#dd6b20;">**현직쌤 팁**</span>: "schwa는 왜 강세를 받을 수 없는가?"를 한 문장으로 설명할 수 있어야 한다 — "Schwa [ə] is a fully reduced, weightless vowel whose rhyme does not branch; therefore it cannot attract primary stress."

---

### 2) N/Adj Stress vs Verb Stress

> For nouns and adjectives: if the penult is heavy, stress falls on the penult; if the penult is light (including schwa), stress falls on the antepenult. For verbs: if the ultimate is heavy, stress falls on the ultimate; if the ultimate is light, stress falls on the penult.

<span style="color:#3182ce;">**N/Adj Stress Rule**</span>과 <span style="color:#3182ce;">**Verb Stress Rule**</span>의 결정적 차이는 **어디서부터 판별하느냐**다. 명사/형용사는 **penult → antepenult** 순으로, 동사는 **ult → penult** 순으로 판별한다.

| 품사 | 1차 판별 | 결과 |
|------|----------|------|
| N/Adj | penult heavy? → penult 강세 | a**gen**da, to**ma**to |
| N/Adj | penult light(ə)? → antepenult 강세 | **ca**lendar, **a**nimal |
| V | ult heavy? → ult 강세 | de**feat**, ar**rive** |
| V | ult light? → penult 강세 | **o**pen, **vis**it |

---

### 3) N–V Homograph Pairs — 최빈출 단골

> When the same word functions as a noun, stress falls on the penult (typically the first syllable); when it functions as a verb, stress falls on the ultimate (typically the second syllable). The unstressed syllable undergoes **vowel reduction** to [ə], making the two pronunciations distinctly different.

<span style="color:#3182ce;">**N–V Homograph Pairs**</span>는 동일 철자의 단어가 품사에 따라 강세 위치가 달라지는 쌍이다. <span style="color:#c53030;">**중요**</span>: 강세 위치만 바뀌는 게 아니라, 강세를 잃은 음절의 모음이 [ə]로 약화(<span style="color:#3182ce;">**vowel reduction**</span>)되어 발음이 상당히 달라진다.

| 단어 | N 강세 + IPA | V 강세 + IPA | 약화 음절 |
|------|-------------|-------------|-----------|
| permit | **ˈper**mit [ˈpɜːrmɪt] | per**ˈmit** [pərˈmɪt] | pɜːr → pər |
| record | **ˈre**cord [ˈrekərd] | re**ˈcord** [rɪˈkɔːrd] | re → rɪ |
| conduct | **ˈcon**duct [ˈkɒndʌkt] | con**ˈduct** [kənˈdʌkt] | kɒn → kən |
| protest | **ˈpro**test [ˈproʊtɛst] | pro**ˈtest** [prəˈtɛst] | proʊ → prə |
| present | **ˈpre**sent [ˈprɛzənt] | pre**ˈsent** [prɪˈzɛnt] | prɛ → prɪ |
| object | **ˈob**ject [ˈɒbdʒɛkt] | ob**ˈject** [əbˈdʒɛkt] | ɒb → əb |
| suspect | **ˈsus**pect [ˈsʌspɛkt] | sus**ˈpect** [səˈspɛkt] | sʌ → sə |
| export | **ˈex**port [ˈekspɔːrt] | ex**ˈport** [ɪkˈspɔːrt] | ek → ɪk |

> 💡 <span style="color:#dd6b20;">**현직쌤 팁**</span>: 답안에 "명사는 penult, 동사는 ult" + "비강세 음절의 vowel reduction(→ schwa)"을 반드시 세트로 기술하라. homograph이지만 homophone은 아니다(homograph but **not** homophonous) — 이 구분도 자주 출제된다.

### 4) Suffix Stress Types — bearing / shifting / neutral 3분류

> English suffixes fall into three classes with respect to stress: **stress-bearing** suffixes attract stress to themselves; **stress-shifting** (stress-fixing) suffixes pull stress to the syllable immediately preceding them; **stress-neutral** suffixes leave the base stress entirely unchanged.

<span style="color:#3182ce;">**suffix stress**</span>의 3분류는 파생어 체인 분석의 뼈대가 된다.

**A. Stress-bearing suffixes** — 접미사 자체가 heavy syllable이어서 강세를 끌어당긴다.

| 접미사 | 예 | 비고 |
|--------|----|----|
| **-ation / -tion** | realize → realiz**a**tion | 가장 빈출 |
| **-ee** | absent → absent**ee** | 예외: **ˈ**committee |
| **-eer** | mountain → mountain**eer** | closed → heavy |
| **-ese** | Japan → Japan**ese** | |
| **-ette** | kitchen → kitchen**ette** | |
| **-ade** | lemon → lemon**ade** | |

**B. Stress-shifting (stress-fixing) suffixes** — 접미사 자체는 강세를 안 받지만, 강세를 **접미사 바로 앞 음절**로 고정한다.

| 접미사 | 예 | 주의 |
|--------|----|----|
| **-ic** | ˈperiod → peri**ˈo**dic | <span style="color:#c53030;">예외: ˈArabic, ˈlunatic, ˈheretic, ˈcatholic</span> |
| **-ify** | ˈperson → per**ˈso**nify | 항상 접미사 직전 음절 |
| **-ity** | ˈhumid → hu**ˈmi**dity | |
| **-ical** | ge**ˈo**metry → ge**ˈo**metrical | -al은 neutral이지만 -ical은 shifting |
| **-ious** | ˈlabor → la**ˈbo**rious | |
| **-ian** | ˈlibrary → li**ˈbra**rian | |
| **-y (명사형)** | ˈphoto → pho**ˈto**graphy | 형용사형 -y(silky)는 neutral — 반드시 구분! |

**C. Stress-neutral suffixes** — 어근의 강세 위치를 전혀 바꾸지 않는다. 8개 굴절 접미사(-s, -ed, -ing, -er, -est 등)는 전부 neutral. 파생 접미사 중에서도 -ness, -ment, -ful, -less, -ly, -able, -er, -ism, -ist, -al 등이 여기 속한다.

> <span style="color:#c53030;">**오개념 주의**</span>: -ness, -ment, -able을 shifting으로 착각하는 경우가 많다. 이 세 접미사는 모두 neutral — ˈhappy → ˈhappiness, aˈmend → aˈmendment, ˈcomfort → ˈcomfortable.

### 5) Compound Stress vs NP Stress

> In **compound words**, primary stress falls on the first element, regardless of meaning. In **noun phrases**, stress falls on the head (the second element). The key distinguishing factor is semantic specialization — a compound denotes a single concept distinct from the literal combination of its parts.

<span style="color:#3182ce;">**compound stress**</span>와 <span style="color:#3182ce;">**NP stress**</span>의 구분은 **의미의 특수화(semantic specialization)** 여부로 판별한다.

| | 예 | 강세 | 의미 |
|-|----|----|------|
| Compound | **ˈblack**bird | 첫 요소 | 찌르레기 (특정 종) |
| NP | black **ˈbird** | 두 번째 요소 | 검은색 새 (묘사) |
| Compound | **ˈgreen**house | 첫 요소 | 온실 (특수화) |
| NP | green **ˈhouse** | 두 번째 요소 | 초록색 집 |
| Compound | **ˈWhite** House | 첫 요소 | 백악관 |

예외: Particle + V 구조(undertake, oversleep)는 **두 번째 요소(V)에 강세** — 유일한 예외 패턴.

---

### 6) Derivation Chains — photograph 패러다임

> Derivation chains track how stress shifts as suffixes are added one by one. Each suffix must be classified as bearing, shifting, or neutral; the final stress position is determined by the outermost suffix's rule applied to the current form.

<span style="color:#3182ce;">**Derivation chain**</span> 분석은 접미사를 하나씩 붙이면서 그때마다 bearing/shifting/neutral을 판별해 최종 강세 위치를 추적하는 것이다.

**photograph 체인 — 임용 최빈출**

| 단계 | 단어 | IPA | 적용 규칙 |
|------|------|-----|-----------|
| 어근 | **ˈpho**tograph | ˈfoʊ.tə.ɡræf | penult = ə → antepenult |
| + -y(명사형) | pho**ˈto**graphy | fə.ˈtɒɡ.rə.fi | -y(명사) = shifting → 접미사 앞 |
| + -ic | photo**ˈgraph**ic | ˌfoʊ.tə.ˈɡræf.ɪk | -ic = shifting → 접미사 앞 + ˌ이차 강세 |
| + -er(중립) | pho**ˈto**grapher | fə.ˈtɒɡ.rə.fər | -er = neutral → photography 강세 유지 |

**economy 체인**

| 단계 | 단어 | IPA | 적용 규칙 |
|------|------|-----|-----------|
| 어근 | e**ˈco**nomy | ɪ.ˈkɒn.ə.mi | -y(명사) = shifting |
| + -ic | eco**ˈno**mic | ˌiː.kə.ˈnɒm.ɪk | -ic = shifting + ˌ이차 강세 |
| + -al | eco**ˈno**mical | ˌiː.kə.ˈnɒm.ɪ.kəl | -al = neutral |
| + -ally | eco**ˈno**mically | ˌiː.kə.ˈnɒm.ɪk.li | -ally = neutral |

**democrat 체인**

| 단계 | 단어 | IPA | 적용 규칙 |
|------|------|-----|-----------|
| 어근 | **ˈde**mocrat | ˈdem.ə.kræt | penult = ə → antepenult |
| + -y(명사) | de**ˈmo**cracy | dɪ.ˈmɒk.rə.si | -y(명사) = shifting |
| + -ic | demo**ˈcrat**ic | ˌdem.ə.ˈkræt.ɪk | -ic = shifting + ˌ이차 강세 |
| + -ally | demo**ˈcrat**ically | ˌdem.ə.ˈkræt.ɪk.li | -ally = neutral |

## ② 기출 맥락

Stress 단원에서 가장 자주 출제된 포인트는 다음 세 가지다.

**N-V Homograph Pairs**: "다음 단어를 명사/동사로 발음할 때 강세와 모음 변화를 IPA로 쓰시오" 형태. 강세 이동뿐 아니라 vowel reduction(약화 음절의 schwa화)까지 묻는다. permit, record, conduct, protest, present 등이 단골 소재.

**photograph / economy 체인**: "다음 파생어들의 강세 위치를 IPA로 표시하고, 적용된 suffix stress 규칙을 서술하시오" 형태. 접미사별 규칙(bearing/shifting/neutral) 판별 + IPA 전사를 세트로 요구한다.

**suffix 분류 판별**: "-ic가 붙으면 강세가 어떻게 이동하는가?" / "다음 중 stress-neutral suffix는?" 형태로 출제. <span style="color:#c53030;">**-ness, -ment를 shifting으로 오답 쓰는 경우**</span>가 많아 neutral임을 명확히 해야 한다.

---

## ③ 키텀 비교

**stress-shifting vs stress-bearing**

가장 많이 혼동하는 쌍이다. shifting은 강세를 **접미사 바로 앞 음절**로 끌어당기고(-ic, -ify, -ity, -y명사형), bearing은 강세를 **접미사 자체**로 끌어당긴다(-ation, -ee, -eer). 쉽게 구분하는 법: 접미사에 강세 기호가 오면 bearing, 접미사 **앞** 음절에 오면 shifting.

**compound vs NP**

의미 특수화(semantic specialization)가 핵심 판별 기준이다. <span style="color:#3182ce;">**ˈblackboard**</span>(칠판 — compound)와 **black ˈboard**(검은색 판자 — NP)처럼, 합성어는 두 요소의 literal 의미를 벗어난 개념을 지시한다. 강세 패턴만 보고 "첫 음절 강세 = compound"로 단정 짓지 말고, **의미 특수화 여부**를 먼저 확인해야 한다.

**형용사형 -y vs 명사형 -y**

<span style="color:#c53030;">**반드시 구분**</span>: 형용사를 만드는 -y(silky, cloudy, windy)는 **stress-neutral**이고, 명사를 만드는 -y(photography, democracy, economy)는 **stress-shifting**이다. 같은 -y지만 기능이 다르면 규칙이 달라진다.

**homograph vs homophonous**

N-V pair는 철자가 같으므로 <span style="color:#319795;">**homograph**</span>다. 그러나 강세와 vowel reduction으로 발음이 달라지므로 <span style="color:#c53030;">**homophonous(동음이의)는 아니다**</span>. 이 구분이 답안 서술에서 중요하다.

## ④ 수험 팁

> 💡 <span style="color:#dd6b20;">**현직쌤 팁 — Derivation chain 답안 작성법**</span>

파생어 체인 문제는 **단계별로 접미사 → 규칙 → 결과를 추적**하는 형식으로 쓴다:

```
photograph [ˈfoʊ.tə.ɡræf]
+ -y (stress-shifting, 명사형) → photography [fə.ˈtɒɡ.rə.fi]
   (접미사 바로 앞 음절 /tɒɡ/에 강세 이동)
+ -ic (stress-shifting) → photographic [ˌfoʊ.tə.ˈɡræf.ɪk]
   (접미사 바로 앞 음절 /ɡræf/에 강세 이동, 첫 음절에 이차 강세 ˌ)
+ -er (stress-neutral) → photographer [fə.ˈtɒɡ.rə.fər]
   (photography의 강세 위치 그대로 유지)
```

> 💡 <span style="color:#dd6b20;">**현직쌤 팁 — -ic 예외 암기**</span>

**ˈArabic / ˈlunatic / ˈheretic / ˈcatholic / ˈpolitic** — 이 단어들은 -ic가 붙었어도 역사적 이유로 첫 음절 강세를 유지한다. "다음 중 -ic가 붙어도 shifting이 일어나지 않는 단어는?" 형태로 출제되므로 5개 외워두자.

> 💡 <span style="color:#dd6b20;">**현직쌤 팁 — Schwa 판별 훈련**</span>

penult에 schwa가 있는지 확인하는 가장 빠른 방법은 **실제로 소리내어 발음**해보는 것이다. 약하게 발음되는 음절(ə)은 귀로 들리고, 그 음절이 penult면 antepenult 강세다. 시험장에서 확신이 없으면 schwa 여부부터 체크하라.

> 💡 <span style="color:#dd6b20;">**현직쌤 팁 — N/Adj vs V 규칙 비교 표**</span>

| | 판별 시작점 | 1순위 강세 | 2순위 강세 |
|-|-----------|-----------|-----------|
| N/Adj | penult | penult heavy → penult | penult light → antepenult |
| V | ult | ult heavy → ult | ult light → penult |

이 표 하나만 기억하면 어떤 단어든 품사 알면 강세 판별 가능하다.
