## 드릴 1 — compound vs NP 구별

다음 표현이 compound이면 C, noun phrase이면 NP로 표시하고, 강세 위치를 ˈ로 표기하시오.

| 표현 | 의미 | C/NP | 강세 표기 |
|------|------|------|----------|
| greenhouse (온실) | | | |
| green house (녹색 집) | | | |
| blackboard (칠판) | | | |
| black board (검은 판) | | | |
| hotdog (핫도그) | | | |
| hot dog (뜨거운 개) | | | |
| darkroom (암실) | | | |
| dark room (어두운 방) | | | |

<details>
<summary>정답 확인</summary>

| 표현 | C/NP | 강세 표기 |
|------|------|----------|
| greenhouse (온실) | C | **gréen**house |
| green house (녹색 집) | NP | green **hóuse** |
| blackboard (칠판) | C | **bláck**board |
| black board (검은 판) | NP | black **bóard** |
| hotdog (핫도그) | C | **hót**dog |
| hot dog (뜨거운 개) | NP | hot **dóg** |
| darkroom (암실) | C | **dárk**room |
| dark room (어두운 방) | NP | dark **róom** |

</details>

## 드릴 2 — NSR 적용 연습

다음 NP/문장에서 NSR에 따라 핵 강세를 받는 단어를 고르고, 강세 표기를 하시오.

1. a tall **?** → beautiful woman
2. She read a **?** → interesting book
3. The cat sat on the **?** → mat
4. He bought a red **?** → sports car
5. university **?** → graduate student

<details>
<summary>정답 및 해설</summary>

1. beautiful **wóman** — 가장 오른쪽 내용어(N)인 woman에 NSR
2. an interesting **bóok** — 가장 오른쪽 내용어 book
3. The cat sat on the **mát** — 가장 오른쪽 내용어 mat
4. a red sports **cár** — 가장 오른쪽 N인 car
5. university graduate **stúdent** — 가장 오른쪽 N인 student

</details>

## 드릴 4 — 함정 판단 (O/X)

1. *Whitehouse* (백악관)에서 강세는 두 번째 요소 *house*에 온다. ( )
2. compound verb *undertake*는 두 번째 요소 *take*에 강세가 온다. ( )
3. NP *a green house* (녹색 집)에서 NSR에 따라 *house*에 강세가 온다. ( )
4. complex compound *high school student*는 *student*에 주강세가 온다. ( )
5. semantic specialization이 있으면 compound로 판단한다. ( )

<details>
<summary>정답 및 해설</summary>

1. **X** — Whitehouse는 compound이므로 첫 요소 **Whíte**house
2. **O** — particle/adverb + V 구조(under + take)는 예외적으로 두 번째 요소에 강세
3. **O** — NP에는 NSR 적용 → 가장 오른쪽 내용어 **hóuse**
4. **X** — complex compound는 첫 요소에 강세 → **hígh** school student
5. **O** — compound는 의미론적으로 구성 요소의 단순 합 이상의 의미(semantic specialization)를 가진다

</details>

## 드릴 5 — 강세 표기 + 의미 구별 종합

다음 두 문장의 밑줄 친 부분의 강세를 표기하고, 의미 차이를 한 문장으로 설명하시오.

**A.** She works in a **greenhouse**.
**B.** She lives in a **green house**.

<details>
<summary>모범 답안</summary>

- A: **gréen**house — compound; 온실(식물 재배용 유리 건물)이라는 특수한 의미
- B: green **hóuse** — NP + NSR; 단순히 녹색으로 칠해진 집

강세가 첫 요소에 오면 compound (특수 의미), 마지막 요소에 오면 NP (합성 의미)로 구별한다.

</details>

---

<span style="color:#dd6b20;">**쌤 팁**</span> — 드릴을 마친 후 이 한 줄만 기억하자: **compound → 첫 요소 강세 / NP → NSR (마지막 내용어)**. 예외는 particle+V compound (두 번째 요소)와 고유명사 기반 compound 정도만 챙기면 된다.
