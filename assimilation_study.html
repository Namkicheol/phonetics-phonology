🔤 ① Assimilation 정의 & 동기 ★ 기본개념 ▼
1) 정의
Assimilation은 인접한 음들이 서로를 닮아 더 유사해지는(become more similar) 음운 현상이다. 동화를 통해 조음 위치·유성성·조음 방법 중 하나 이상의 feature가 공유된다.

동화가 일어나는 근본 동기는 ease of articulation(조음 편의)이다 — 인접 음의 특성을 미리 준비하거나 유지하는 방향으로 조음이 효율화된다.

동화는 영어에서 매우 일상적이며, 연음(connected speech)에서 특히 두드러진다.

Dissimilation은 반대 방향 — 인접 음들이 서로 달라지는 현상. 동화와 반드시 구별할 것.

2) 분류 체계
방향 (Direction)
Progressive Regressive 앞→뒤 / 뒤→앞
정도 (Degree)
Total (완전 동화) Partial (부분 동화) 동일 / 일부 feature 공유
인접성 (Adjacency)
Adjacent (인접 동화) Non-adjacent (비인접) 바로 옆 / 건너뛰기
↔️ ② 방향 — Progressive vs Regressive ★★★ 최빈출 ▼
핵심 구분
Progressive Assimilation (순행 동화)
앞 음이 뒤 음에 영향을 줌 방향: 조건음 → 피동화음 (앞→뒤) Conditioning sound precedes 영어에서 상대적으로 드묾 cats [kæts]: /t/ (voiceless) → -s [s]
dogs [dɔɡz]: /ɡ/ (voiced) → -s [z]
Regressive Assimilation (역행 동화)
뒤 음이 앞 음에 영향을 줌 방향: 피동화음 ← 조건음 (뒤→앞) Conditioning sound follows 영어에서 더 흔함 ten boys [tɛm bɔɪz]: /b/ → /n/→[m]
in+possible → impossible: /p/ → /n/→[m]
⚠️ 시험장 확인 포인트
영어의 Place assimilation은 Regressive. "뒤 자음의 조음 위치가 앞의 /n/을 동화시킨다"는 방향이 핵심 출제 포인트 (2010 Q31 직접 출제). Progressive와 혼동하면 오답.
역할 용어 — Trigger vs Target

Trigger (조건음, conditioning sound): 동화를 유발하는 음

Target (피동화음, assimilated sound): 동화되는 음

Regressive에서: Trigger = 뒤 음, Target = 앞 음

Progressive에서: Trigger = 앞 음, Target = 뒤 음

📋 기출 이력

2010 Q31 — Place assimilation의 방향이 regressive임을 확인하는 문항. /n/ → [m]/[ŋ] 환경에서 trigger(뒤 자음)와 target(앞 /n/) 구별.

📊 ③ 정도 — Total vs Partial + 인접성 ★ 기본개념 ▼
정도(Degree)에 따른 분류
Total Assimilation (완전 동화)
동화 결과 두 음이 완전히 동일해짐 → gemination (겹자음화)이라고도 함 give me → [ɡɪmmi]
let me → [lɛmmi]
good boy → [ɡʊbbɔɪ]
pet cat → [pɛkkæt]
Partial Assimilation (부분 동화)
인접 음의 일부 feature만 공유 → 동화 실패가 아니라 정상적 결과 ten boys → [tɛm bɔɪz]
/n/ → [m]: nasal 유지 + place만 bilabial로 변화
(→ 두 음은 여전히 다름: [m] ≠ [b])

방향(Progressive/Regressive)과 정도(Total/Partial)는 독립적인 분류 차원이다 — 동시에 적용됨.

인접성(Adjacency)에 따른 분류

Adjacent assimilation: 조건음과 피동화음이 바로 인접. 성인 영어의 일반적인 형태.

Non-adjacent assimilation: 사이에 다른 음이 끼어 있는 경우 (consonant harmony). 어린이 발화에서 흔함.

dog → [ɡɔɡ]  (d → g, initial assimilates to final — children)
sun → [nʌn]  (s → n, initial assimilates to final — children)
📍 ④ Place Assimilation — 조음 위치 동화 ★★★ 최빈출 ▼
1) 정의 & 핵심 원리
Place assimilation: 한 음이 인접 음의 조음 위치(place of articulation) feature를 닮아가는 동화.

영어에서는 대부분 Regressive — 뒤에 오는 자음의 조음 위치가 앞에 오는 음(주로 /n/)을 동화시킨다.
2) Nasal Place Assimilation — /n/ 의 세 가지 실현
Phonological Rule
/n/ → [m] / ___ {bilabial: p, b}
/n/ → [ŋ] / ___ {velar: k, g}
/n/ → [n] / elsewhere (alveolar 앞 또는 기타 환경)
ten boys
/n/ → [m]
/b/ bilabial
ten people
/n/ → [m]
/p/ bilabial
ten girls
/n/ → [ŋ]
/g/ velar
ten keys
/n/ → [ŋ]
/k/ velar
ten cups
/n/ → [ŋ]
/k/ velar

방향: Regressive — 뒤의 /b,p/ 또는 /g,k/가 앞의 /n/을 동화시킴.

정도: Partial — /n/ → [m]: nasal manner 유지 + place만 bilabial로 변화.

3) iN- prefix — Morphophonemic Rule
부정 접두사 iN-의 /N/은 뒤에 붙는 어근의 첫 자음의 조음 위치에 동화된다.
접두사 형태환경 (뒤 자음)예시
im- [ɪm]bilabial /p, b, m/ 앞impossible, imbalance, immature
in- [ɪn]alveolar /t, d, n, s, l/ 앞 (기본형)indirect, indirect, intolerant
iŋ- [ɪŋ]velar /k, g/ 앞inconclusive [ɪŋ-], ingratitude [ɪŋ-]
il- [ɪl]/l/ 앞 (Manner assimilation)illegal, illogical, illiterate
ir- [ɪɹ]/r/ 앞 (Manner assimilation)irregular, irrational, irrecoverable

il-/ir-는 place assimilation이 아니라 manner assimilation (→ 섹션 ⑥ 참조).

⚠️ 주의
iN+velar 동화 조건: stress가 걸린 /n/ 음절 바로 앞에서만 적용됨 (e.g., inˈconclusive에서 /n/이 강세 받는 위치). 강세 없는 환경에서는 적용 안 될 수 있음.
4) Stop 자음 Place Assimilation (문장 경계)
단어 경계에서 alveolar stops /t, d/가 뒤에 오는 bilabial 또는 velar stop의 조음 위치에 동화된다.
good boy
/d/→[b]
[ɡʊbbɔɪ]
night cap
/t/→[k]
[naɪkkæp]
white paper
/t/→[p]
[waɪppɛɪpɚ]
red badge
/d/→[b]
[rɛbbædʒ]

voicing은 유지된 채 place만 바뀜 → partial assimilation.

📋 기출 이력

2010 Q31 — 조음 위치 동화 규칙 적용: /n/의 place assimilation 방향(regressive) 판별.

2017 기입1 — nasal assimilation 규칙 환경 파악: in+possible → impossible. /n/ → [m] / __ [p] (bilabial).

🔊 ⑤ Voice (Laryngeal) Assimilation — 유성성 동화 ★★ 빈출 ▼
1) 정의
Voice assimilation (= Laryngeal assimilation): 한 음이 인접 음의 유성성(voicing) feature를 닮아가는 동화.

영어에서 대부분 Progressive — 앞에 오는 자음의 유성성이 뒤의 굴절 형태소에 영향을 미침.
2) 복수 형태소 -s: [s] / [z] / [əz]
Rule — Progressive Voice Assimilation
-s → [s] / voiceless segment ___ (무성음 뒤)
-s → [z] / voiced segment ___ (유성음 뒤)
-s → [əz] / sibilant ___ (sibilant 뒤 — schwa insertion)
cats [s]
/t/ voiceless
taps [s]
/p/ voiceless
dogs [z]
/g/ voiced
cabs [z]
/b/ voiced
buses [əz]
/s/ sibilant
3) 과거 형태소 -ed: [t] / [d] / [əd]
Rule — Progressive Voice Assimilation
-ed → [t] / voiceless segment ___ (무성음 뒤)
-ed → [d] / voiced segment ___ (유성음 뒤)
-ed → [əd] / alveolar stop /t,d/ ___ (alveolar stop 뒤)
walked [t]
/k/ voiceless
pushed [t]
/ʃ/ voiceless
robbed [d]
/b/ voiced
seemed [d]
/m/ voiced
tested [əd]
/t/ alveolar
blended [əd]
/d/ alveolar
4) 어휘 내 Voice Assimilation — have to / has to / used to

어휘 내에서도 유성성 동화 적용: 뒤에 오는 /t/(voiceless)가 앞 음을 무성화.

have to
/v/ → [f] before /t/
[hæftə]
has to
/z/ → [s] before /t/
[hæstə]
used to
/zd/ → [st]
[juːstə]

이 경우는 Regressive voice assimilation — 뒤의 /t/(voiceless)가 앞 음을 무성화.

5) Sonorant Devoicing

초성 자리의 sonorant(/l, r, w, j/)는 어두 무성 stop (/p, t, k/) 뒤에서 부분 무성화된다 (progressive voice assimilation).

play [pl̥eɪ]
try [tɹ̥ɑɪ]
twin [tw̥ɪn]
📋 기출 이력

2015 기입1 — 유성음화 동화 규칙 적용: 복수 -s, 과거 -ed의 [s]/[z], [t]/[d] 교체 조건 판별.

2011 Q30 — 음운 규칙과 동화 현상: voice assimilation 포함 전반 음운 규칙 적용.

🌊 ⑥ Manner Assimilation — 조음 방법 동화 ★★ 빈출 ▼
1) 정의
Manner assimilation: 한 음이 인접 음의 조음 방법(manner of articulation) feature를 닮아가는 동화.
2) Vowel Nasalization (모음 비음화)

모음이 뒤에 오는 비음 /m, n, ŋ/ 앞에서 비음화된다 — manner(nasal) feature가 모음에 전파.

방향: Regressive — 뒤의 nasal이 앞 모음에 영향.

band [bæ̃nd]
/æ/ → [æ̃] before /n/
farm [fɑ̃ːrm]
/ɑ/ → [ɑ̃] before /m/
can [kæ̃n]
/æ/ → [æ̃] before /n/
3) Liquid Manner Assimilation — iN- prefix il- / ir-
부정 접두사 iN-에서 /N/이 어근 첫 자음이 liquid(/l/ 또는 /r/)일 때 해당 liquid에 동화된다.
Rule
/N/ → [l] / ___ [l] → il-: illegal, illogical, illiterate
/N/ → [r] / ___ [r] → ir-: irregular, irrational, irrecoverable
/in + legal/
→
/n/ → [l] (liquid assimilation)
→
[ɪlliːɡəl]
/in + regular/
→
/n/ → [r] (liquid assimilation)
→
[ɪrreɡjələr]

이것은 place가 아니라 manner 동화 — /n/(nasal stop) → [l]/[r](liquid)으로 조음 방법 자체가 바뀜.

⚠️ 구별 포인트
im-/iŋ-는 Place assimilation (조음 위치), il-/ir-는 Manner assimilation (조음 방법). 같은 iN- prefix에서 서로 다른 동화 유형이 작동한다.
📋 기출 이력

2020 B-3 — il-/ir- 분포 조건: 어근의 첫 자음이 /l/이면 il-, /r/이면 ir- 선택. Liquid manner assimilation 원리 적용.

2011 Q30 — 음운 규칙과 동화 현상 전반 (liquid assimilation 포함).
